# RSD v0.20 – Invoice PDF Generation

## Goal

Generate professional PDF invoices for students.

## Invoice Number Format

RSD-INV-YYYY-XXXXXX

Example:
RSD-INV-2026-000001

## Invoice Should Include

* Ready Set Drive logo
* Invoice number
* Invoice date
* Student details
* Lesson date and time
* Lesson description
* Amount due
* Payment status
* Payment reference (if paid)
* Ready Set Drive business details

## Lesson Description Rules

### Standard Lesson

Display:
Automatic Driving Lesson

### Test Booking

Display:
Practical Driving Test Package

## Invoice Button

Always visible regardless of payment status.

Reason:

* Reprint invoices
* Generate invoice before payment
* Generate invoice after payment
* Lost paperwork replacement

## Phase 1 Scope

* Generate invoice from a single lesson
* Professional PDF layout
* Download PDF manually
* Invoice numbering
* Payment status display

## Future Enhancements

### Phase 2

* Multi-lesson invoices
* Invoice history
* Store invoice records

### Phase 3

* Email invoices
* Student portal integration
* Automatic invoice reminders
